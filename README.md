# 💱 Currency Converter (Python)

A modern Currency Converter application built using Python and CustomTkinter.  
This app converts currencies in real-time using an online API and provides a clean GUI experience.

---

## 🚀 Features

- 🌍 Convert between multiple currencies
- 🔄 Swap currencies instantly
- 💡 Live exchange rate display
- 💳 Transaction fee calculation
- 🎨 Dark / Light mode
- 🧮 Built-in calculator
- 🔍 Search currency by name
- 🏳️ Country flags display

---

## 🛠️ Technologies Used

- Python
- CustomTkinter (GUI)
- Requests (API)
- Pillow (Images)

---

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/Tusharrajdev/currency-converter-python.git
2. Go to project folder:
```bash
cd currency-converter-python
```

3. Install dependencies:
```bash
pip install customtkinter requests pillow
```

4. Run the app:
```bash
python main.py
```
